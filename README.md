<!-- GETTING STARTED -->
## Getting Started

To get a local copy up on your own computer and running follow these simple steps.

### Prerequisites

1. yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rachelim21/PolkaPay.git
   ```
2. Run the React server
    ```sh
    cd polkapay
    yarn start
    ```
